Welcome to the antaris test page

What is antaris?

Antaris is the prototype of a php based application server, similar to those known in the Java world, like Spring.

How does it work?

Antaris runs php as a standalone process via php-cli. It relies on apache2 to handle incoming http and https traffic.

To forward the traffic from apache2 to antaris, mod_jk is used, which is the Apache Jakarta module, used for Java based Tomcat instances.

Jakarta relies on the AJPv13 protocol, which has been adapted to php. That is the essence of the antaris' core - a library for parsing AJP13 packets and convert them into php objects to work with.

Why mod_jk?

To avoid the complex task of parsing HTTP requests in php, we're using apache2 with mod_jk. Using this approach, we can rely on the experience of the Apache Foundation's projects and their experience. This also saves as from a lot of headaches related to security issues.

Now, can I test its capabilities?

Actually, you're doing it right now! This page runs on an antaris instance, in a Docker container. Give it a try and check out the various test pages we've prepared for you. Make sure to take a look at the Image to HTML converter, which will convert your uploaded image into HTML/CSS and render it.
